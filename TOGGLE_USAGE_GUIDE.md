## 🐛 故障排除

### 问题1：看不到开关

**可能原因：**
- VS Code Webview缓存未更新
- 扩展未重新加载

**解决方法：**
```
1. Ctrl+Shift+P → "Developer: Reload Window"
2. 重新打开 CodeYang: Start Chat
3. 如果还是不显示，完全重启VS Code
```

### 问题2：点击开关没反应

**检查步骤：**
1. 打开浏览器开发者工具（右键 → "检查"）
2. 查看Console是否有JavaScript错误
3. 确认 `toggleToolCollapse()` 函数存在

### 问题3：设置不保存

**原因：** localStorage被禁用或不可用

**解决：**
1. 检查VS Code的webview设置
2. 确认没有隐私插件阻止localStorage
3. 尝试在测试页面中验证
