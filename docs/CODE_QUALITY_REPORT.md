### 1. 裸 catch 块泛滥 (严重) - 扣 10 分
**问题:**
- ❌ 133 个裸 `catch {}` 块
- ❌ 异常被静默吃掉，出错难以定位
- ❌ 分布在 67 个文件中

**影响:**
- 调试困难
- 错误被隐藏
- 生产问题难以追踪

**示例:**
```typescript
// 危险模式
try {
  await criticalOperation();
} catch {
  // 什么都不做，错误被吞掉
}

// 应该改为
try {
  await criticalOperation();
} catch (err) {
  console.error('[Module] Operation failed:', err);
  // 适当的降级处理
}
```

**已修复:** 7/133 (Agent.ts)
**待修复:** 126
